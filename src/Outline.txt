Java Class Creator Outline:
**********
This is a description for how to structure your outline.
This is NOT to be used to test your program. When testing use ExampleOutline.txt
**********

Project Name
{ClassName
	{Fields
		<access> <static> <type> <name>
			access default - private. non-static unless specified.
		...
		...
	}
	{Methods
		Default getter/setters for every variable name. Don't need to include them in the text file.
		<access> <static> <return> <signature>
			access default - public. non-static unless specified.
		...
		...
	}
}
{Class2Name
	{Fields
		...
	}
	{Methods
		...
	}
}
